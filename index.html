Features

Segmentation & Rule Engine

Optimize better by defining targeted configurations and hierarchies for your entity segments using the segmentation tree. Define more complex rules in the form of logical expressions using the rule engine.

Constraints and Validations

Never run into the problem of misbehaviour in your applications due to incorrect configuration values with data type validation and constraints check like minimum and maximum value and length, patterns, etc.

Dashboards With ACLs And Audits

Never run into unauthorized configuration changes by grouping related set of configurations into collections and providing fine grained access controls. Keep an audit of when configuration changes were made and who made them.

Manage Multiple Environments

Manage configurations for all your environments like production, beta, staging and other lower environments from a single instance of Controllable.

Scalable With Load

Scale Controllable config nodes horizontally to serve massive amounts of traffic. Currently Controllable is offered with a licensed self hosted version with a managed hosting planned in the future. Host Controllable close to your existing infra and edge.

SDKs for Multiple Languages

Support for multiple languages like Python, Golang, Java, Javascript and Ruby available. Additional languages support is being added, but in the meanwhile, the REST interface can be used for unsupported languages.
